# SpartaTown

## 프로젝트 소개

Unity를 사용한 스파르타 타운 (게더 클론)

## 개발 기간

- 22.11.24 ~ 22.11.29

## 구현된 기능

#### 1. 캐릭터
   - wasd와 방향키를 사용하여 움직임
   - 화면은 캐릭터를 중심으로 움직임
   - 캐릭터는 마우스 포인터를 바라봄
   - 캐릭터 애니메이션
   
#### 2. 맵
   - 벽과 충돌하며 넘어가지 못함

#### 3. 이름 설정
   - 시작화면에서 이름을 설정하면 캐릭터 위에 이름이 생김
   - 2~10글자 사이만 가능
